Directory name to store CVS modules. 

    - workspace
      - directory_location
         - module1
           - src
           - pom.xml
         - module2
           - src
           - pom.xml

Plugin avoids creating a sub-directory inside the workspace in case of one location and one module, i.e. "flat" structure will be used: 

    - workspace
      - src
      - pom.xml
