Строка соединения с CVS, которую будт использовать Hudson. Формат такой же как у переменной окружения $CVSROOT (:protocol:user:password@host:path). Password опционален, если он не был указан Hudson будет использовать данные из .cvspass файла (по умолчанию из user home, это локация может быть изменена на глобальной конфигурационной странице).
